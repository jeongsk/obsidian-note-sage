AI एजेंट्स को सीधे अपने Obsidian vault में लाएं। यह प्लगइन Claude Code CLI को Obsidian के साथ सहजता से एकीकृत करता है - जिससे आप AI के साथ चैट कर सकते हैं, फ़ाइलें संपादित कर सकते हैं, और अपने कार्यक्षेत्र को छोड़े बिना अपने ज्ञान आधार का प्रबंधन कर सकते हैं।

## क्यों?

AI के साथ काम करने से आपका प्रवाह बाधित नहीं होना चाहिए या आपके vault का संदर्भ नहीं खोना चाहिए।

यह प्लगइन Claude को आपके Obsidian वातावरण के बारे में जागरूक रखता है जबकि आपको टर्मिनल पर स्विच करने के बजाय अपने कार्यक्षेत्र में रहने देता है। यह संदर्भ को अधिक प्रभावी ढंग से प्रबंधित करने में मदद करने के लिए सुविधाएं भी प्रदान करता है।

## विशेषताएं

- **प्रत्यक्ष AI एकीकरण**: Obsidian के इंटरफ़ेस में सीधे Claude Code के साथ चैट करें
- **फ़ाइल-प्रथम दृष्टिकोण**: AI एजेंट आपके vault में फ़ाइलें पढ़, संपादित और बना सकते हैं
- **संदर्भ-जागरूक**: बातचीत में स्वचालित रूप से प्रासंगिक vault संदर्भ शामिल करता है
- **रीयल-टाइम सहयोग**: अपने Obsidian इंटरफ़ेस में AI संपादन लाइव देखें
- **त्वरित कार्य**: पूर्वनिर्धारित प्रॉम्प्ट के साथ सामान्य कार्यों को तेज़ी से निष्पादित करें (सारांश, सुधार, विश्लेषण, अनुवाद)
- **MCP सर्वर एकीकरण**: बाहरी उपकरणों और संसाधनों को AI एजेंट से कनेक्ट करें
- **बहुभाषी UI**: 11 भाषाओं के लिए समर्थन (हिंदी, अंग्रेज़ी, कोरियाई, जापानी, चीनी, जर्मन, फ्रेंच, स्पेनिश, पुर्तगाली, रूसी, अरबी)
- **सत्र प्रबंधन**: बातचीत की निरंतरता बनाए रखें और बातचीत सहेजें

## स्थापना

### आवश्यकताएं

- **Anthropic खाता** - Claude API एक्सेस के लिए आवश्यक ([console.anthropic.com](https://console.anthropic.com))
- **Node.js** - प्लगइन निष्पादन के लिए आवश्यक
- **Claude Code CLI** - [Anthropic's Claude Code](https://www.anthropic.com/claude-code) से प्राप्त करें

### सेटअप

*जल्द ही Obsidian कम्युनिटी प्लगइन स्टोर में*

1. **CLI एक्सेस सत्यापित करें**: सुनिश्चित करें कि आप अपने टर्मिनल में `claude` चला सकते हैं
2. **डाउनलोड करें**: [रिलीज़ पेज](../../releases) से नवीनतम रिलीज़ (`obsidian-note-sage.zip`) प्राप्त करें
3. **इंस्टॉल करें**: एक्सट्रैक्ट करें और फ़ोल्डर को `[आपका_vault]/.obsidian/plugins/obsidian-note-sage` में रखें
4. **सक्षम करें**: Obsidian की कम्युनिटी प्लगइन्स सेटिंग्स में प्लगइन को सक्रिय करें
5. **चैटिंग शुरू करें**: अपने कार्यक्षेत्र की दाईं साइडबार में Note Sage पैनल का उपयोग करें

> [!WARNING]
> **पूर्वावलोकन संस्करण सूचना**
>
> यह प्लगइन सक्रिय विकास में है और आपके vault में फ़ाइलों को संशोधित करेगा। यह वर्तमान में पूर्ण कार्यक्षमता के लिए उन्नत अनुमतियों (`--permission-mode bypassPermissions` और `dangerously-skip-permissions`) का उपयोग करता है।
>
> **अनुशंसित**: उपयोग से पहले अपने vault का बैकअप लें। सूक्ष्म अनुमति नियंत्रण भविष्य की रिलीज़ के लिए नियोजित हैं।

## उपयोग

### बुनियादी उपयोग

1. दाईं साइडबार में Note Sage आइकन पर क्लिक करें
2. चैट इनपुट में अपना संदेश टाइप करें
3. भेजने के लिए Enter दबाएं (नई लाइन के लिए Shift+Enter)
4. वर्तमान नोट संदर्भ को शामिल/बाहर करने के लिए "वर्तमान पृष्ठ" बटन का उपयोग करें

### त्वरित कार्य

सामान्य कार्यों को तेज़ी से निष्पादित करने के लिए चैट इनपुट के ऊपर त्वरित कार्य बटन का उपयोग करें:

- **सारांश**: वर्तमान दस्तावेज़ को संक्षेप में सारांशित करें
- **सुधार**: लेखन शैली में सुधार करें और त्रुटियों को ठीक करें
- **विश्लेषण**: दस्तावेज़ का विश्लेषण करें और अंतर्दृष्टि प्रदान करें
- **अनुवाद**: भाषाओं के बीच अनुवाद करें

आप सेटिंग्स में प्रत्येक बटन के प्रॉम्प्ट को अनुकूलित कर सकते हैं।

AI एजेंट की क्षमताओं को विस्तारित करने के लिए MCP (Model Context Protocol) सर्वर कनेक्ट करें।

| सर्वर सक्षम/अक्षम करें | व्यक्तिगत सर्वर टॉगल करें |
| टूल्स पैनल | कनेक्टेड MCP सर्वर से उपलब्ध टूल्स देखें |

## समस्या निवारण

### CLI समस्याएं

**Node.js या Claude CLI नहीं मिल रहा?**

1. **इंस्टॉलेशन सत्यापित करें**: अपने टर्मिनल में `node --version` और `claude --version` चलाएं
2. **मैन्युअल कॉन्फ़िगरेशन**: यदि ऑटो-डिटेक्शन विफल हो जाता है, तो **सेटिंग्स > कम्युनिटी प्लगइन्स > Note Sage** में पथ मैन्युअल रूप से सेट करें

**निष्पादन योग्य पथ खोजें:**
```bash
# Node.js स्थान
which node

# Claude CLI स्थान
echo "$(sed -n 's/^exec "\([^"]*\)".*/\1/p' $(which claude))"
```

**Windows उपयोगकर्ता**: Claude को WSL में इंस्टॉल किया जाना चाहिए। Windows नेटिव समर्थन अभी भी परीक्षण में है।

### डीबग मोड

**विस्तृत लॉगिंग सक्षम करें:**
1. **सेटिंग्स > कम्युनिटी प्लगइन्स > Note Sage** पर जाएं
2. "डीबग मोड" सक्षम करें
3. डेवलपर कंसोल खोलें: `Cmd+Opt+I` (Mac) या `Ctrl+Shift+I` (Windows/Linux)

## गोपनीयता और डेटा हैंडलिंग

### रिमोट सेवाएं

यह प्लगइन आपके अनुरोधों को संसाधित करने के लिए **Anthropic Claude API** का उपयोग करता है। जब आप एक संदेश भेजते हैं:
- आपका प्रॉम्प्ट टेक्स्ट Anthropic के सर्वर पर भेजा जाता है
- यदि "वर्तमान पृष्ठ" संदर्भ सक्षम है, तो फ़ाइल सामग्री भी भेजी जाती है
- डेटा [Anthropic की गोपनीयता नीति](https://www.anthropic.com/privacy) के अनुसार संसाधित किया जाता है

### खाता आवश्यकताएं

- **Anthropic API कुंजी**: API एक्सेस के लिए आवश्यक
  - [console.anthropic.com](https://console.anthropic.com) पर एक प्राप्त करें
  - आपके vault के प्लगइन डेटा में स्थानीय रूप से संग्रहीत (एन्क्रिप्टेड नहीं)

### फ़ाइल सिस्टम एक्सेस

यह प्लगइन आपके Obsidian vault के बाहर स्थानों तक पहुंचता है:
- **Claude CLI डिटेक्शन**: Claude निष्पादन योग्य का पता लगाने के लिए सिस्टम पथों की खोज करता है
  - macOS/Linux: `~/.local/bin/`, `/usr/local/bin/`, `/opt/homebrew/bin/`
  - Windows: `%USERPROFILE%\AppData\`, `C:\Program Files\`
- **एजेंट अनुमतियां**: उन्नत अनुमतियों (`bypassPermissions`) का उपयोग करता है जो AI को आपके vault में फ़ाइलें पढ़ने/संपादित करने की अनुमति देता है

### स्थानीय डेटा संग्रहण

- चैट संदेश केवल मेमोरी में संग्रहीत होते हैं (पुनरारंभ पर साफ़)
- सेटिंग्स `.obsidian/plugins/obsidian-note-sage/data.json` में संग्रहीत
- बातचीत मैन्युअल रूप से आपके vault में markdown फ़ाइलों के रूप में सहेजी जा सकती हैं
