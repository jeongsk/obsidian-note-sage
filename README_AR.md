أضف وكلاء الذكاء الاصطناعي مباشرة إلى مخزن Obsidian الخاص بك. يدمج هذا الإضافة Claude Code CLI بسلاسة مع Obsidian - مما يتيح لك الدردشة مع الذكاء الاصطناعي وتحرير الملفات وإدارة قاعدة معارفك دون مغادرة مساحة عملك.

## لماذا؟

العمل مع الذكاء الاصطناعي لا ينبغي أن يقطع تدفقك أو يفقد سياق المخزن الخاص بك.

تحافظ هذه الإضافة على إدراك Claude لبيئة Obsidian الخاصة بك مع السماح لك بالبقاء في مساحة عملك بدلاً من التبديل إلى المحطة الطرفية. كما توفر ميزات لمساعدتك في إدارة السياق بشكل أكثر فعالية.

## الميزات

- **تكامل مباشر للذكاء الاصطناعي**: تحدث مع Claude Code مباشرة في واجهة Obsidian
- **نهج الملف أولاً**: يمكن لوكلاء الذكاء الاصطناعي قراءة وتحرير وإنشاء الملفات في مخزنك
- **إدراك السياق**: يتضمن تلقائياً سياق المخزن ذي الصلة في المحادثات
- **التعاون في الوقت الفعلي**: شاهد تعديلات الذكاء الاصطناعي تحدث مباشرة في واجهة Obsidian الخاصة بك
- **إجراءات سريعة**: نفذ المهام الشائعة بسرعة باستخدام المطالبات المحددة مسبقاً (تلخيص، تحسين، تحليل، ترجمة)
- **تكامل خادم MCP**: قم بتوصيل الأدوات والموارد الخارجية بوكيل الذكاء الاصطناعي
- **واجهة مستخدم متعددة اللغات**: دعم 11 لغة (العربية، الإنجليزية، الكورية، اليابانية، الصينية، الألمانية، الفرنسية، الإسبانية، البرتغالية، الروسية، الهندية)
- **إدارة الجلسات**: الحفاظ على استمرارية المحادثة وحفظ المحادثات

## التثبيت

### المتطلبات الأساسية

- **حساب Anthropic** - مطلوب للوصول إلى Claude API ([console.anthropic.com](https://console.anthropic.com))
- **Node.js** - مطلوب لتشغيل الإضافة
- **Claude Code CLI** - احصل عليه من [Anthropic's Claude Code](https://www.anthropic.com/claude-code)

### الإعداد

*قريباً في متجر إضافات مجتمع Obsidian*

1. **التحقق من الوصول إلى CLI**: تأكد من أنه يمكنك تشغيل `claude` في المحطة الطرفية الخاصة بك
2. **التنزيل**: احصل على أحدث إصدار (`obsidian-note-sage.zip`) من [صفحة الإصدارات](../../releases)
3. **التثبيت**: استخرج الملفات وضع المجلد في `[مخزنك]/.obsidian/plugins/obsidian-note-sage`
4. **التمكين**: قم بتفعيل الإضافة في إعدادات إضافات المجتمع في Obsidian
5. **ابدأ الدردشة**: استخدم لوحة Note Sage في الشريط الجانبي الأيمن لمساحة عملك

> [!WARNING]
> **إشعار نسخة المعاينة**
>
> هذه الإضافة قيد التطوير النشط وستقوم بتعديل الملفات في مخزنك. تستخدم حالياً أذونات مرتفعة (`--permission-mode bypassPermissions` و `dangerously-skip-permissions`) للحصول على الوظائف الكاملة.
>
> **موصى به**: قم بعمل نسخة احتياطية من مخزنك قبل الاستخدام. من المخطط إضافة عناصر تحكم دقيقة في الأذونات في الإصدارات المستقبلية.

## الاستخدام

### الاستخدام الأساسي

1. انقر على أيقونة Note Sage في الشريط الجانبي الأيمن
2. اكتب رسالتك في مدخل الدردشة
3. اضغط Enter للإرسال (Shift+Enter لسطر جديد)
4. استخدم زر "الصفحة الحالية" لتضمين/استبعاد سياق الملاحظة الحالية

### الإجراءات السريعة

استخدم أزرار الإجراءات السريعة فوق مدخل الدردشة لتنفيذ المهام الشائعة بسرعة:

- **تلخيص**: تلخيص المستند الحالي بإيجاز
- **تحسين**: تحسين أسلوب الكتابة وإصلاح الأخطاء
- **تحليل**: تحليل المستند وتقديم رؤى
- **ترجمة**: الترجمة بين اللغات

يمكنك تخصيص مطالبة كل زر في الإعدادات.

## استكشاف الأخطاء وإصلاحها

### مشاكل CLI

**لا يمكن العثور على Node.js أو Claude CLI؟**

1. **التحقق من التثبيتات**: قم بتشغيل `node --version` و `claude --version` في المحطة الطرفية
2. **التكوين اليدوي**: إذا فشل الاكتشاف التلقائي، قم بتعيين المسارات يدوياً في **الإعدادات > إضافات المجتمع > Note Sage**

**البحث عن مسارات الملفات التنفيذية:**
```bash
# موقع Node.js
which node

# موقع Claude CLI
echo "$(sed -n 's/^exec "\([^"]*\)".*/\1/p' $(which claude))"
```

**مستخدمو Windows**: يجب تثبيت Claude في WSL. الدعم الأصلي لـ Windows لا يزال قيد الاختبار.

### وضع التصحيح

**تمكين التسجيل المفصل:**
1. انتقل إلى **الإعدادات > إضافات المجتمع > Note Sage**
2. قم بتمكين "وضع التصحيح"
3. افتح وحدة تحكم المطور: `Cmd+Opt+I` (Mac) أو `Ctrl+Shift+I` (Windows/Linux)

## الخصوصية ومعالجة البيانات

### الخدمات البعيدة

تستخدم هذه الإضافة **Anthropic Claude API** لمعالجة طلباتك. عند إرسال رسالة:
- يتم إرسال نص المطالبة الخاص بك إلى خوادم Anthropic
- إذا تم تمكين سياق "الصفحة الحالية"، يتم أيضاً إرسال محتويات الملف
- تتم معالجة البيانات وفقاً لـ [سياسة خصوصية Anthropic](https://www.anthropic.com/privacy)

### متطلبات الحساب

- **مفتاح Anthropic API**: مطلوب للوصول إلى API
  - احصل على واحد من [console.anthropic.com](https://console.anthropic.com)
  - يتم تخزينه محلياً في بيانات إضافة المخزن الخاص بك (غير مشفر)

### الوصول إلى نظام الملفات

تصل هذه الإضافة إلى مواقع خارج مخزن Obsidian الخاص بك:
- **اكتشاف Claude CLI**: البحث في مسارات النظام لتحديد موقع ملف Claude التنفيذي
  - macOS/Linux: `~/.local/bin/`، `/usr/local/bin/`، `/opt/homebrew/bin/`
  - Windows: `%USERPROFILE%\AppData\`، `C:\Program Files\`
- **أذونات الوكيل**: يستخدم أذونات مرتفعة (`bypassPermissions`) مما يسمح للذكاء الاصطناعي بقراءة/تحرير الملفات في مخزنك

### تخزين البيانات المحلية

- يتم تخزين رسائل الدردشة في الذاكرة فقط (يتم مسحها عند إعادة التشغيل)
- يتم تخزين الإعدادات في `.obsidian/plugins/obsidian-note-sage/data.json`
- يمكن حفظ المحادثات يدوياً في مخزنك كملفات markdown
